# US Slaughter Dashboard

Interactive Streamlit dashboard visualizing USDA annual slaughter scale (~9.4B animals/yr across species).

## Data Sources
- USDA ERS Cattle & Beef Statistics
- NASS Livestock Slaughter reports (cattle, hogs, sheep)
- Poultry from FAS/ERS

## Run locally
```
streamlit run app.py
```
